# Hello World

A minimal static Hello World site.

Files:
- `index.html` — main page
- `style.css` — minimal styling
- `script.js` — small DOM script for button interaction

Open locally:
- Double-click `index.html` to open in your browser
- Or serve with Python (recommended for fetches):

```bash
# Python 3
cd /Users/josephvo/Desktop/code/hello-world
python3 -m http.server 8000
# then open http://localhost:8000
```
